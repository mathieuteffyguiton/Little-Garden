# Little Garden

Starter kit Godot 4 pour un action-aventure 2D top-down inspiré des premiers Zelda.

## Contenu

- Player top-down
- Stats : niveau, XP, HP, attaque, défense, argent
- Inventaire simple
- Sauvegarde JSON
- Carte du monde simple
- Structure zones / donjons / boss
- Base pour les 10 zones

## Lancement

1. Ouvre Godot 4.
2. Importe le dossier `Little-Garden-Starter`.
3. Lance la scène `scenes/main/Main.tscn`.
