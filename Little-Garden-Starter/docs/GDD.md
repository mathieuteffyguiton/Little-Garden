# Little Garden — Game Design Document

## Genre
Action-aventure 2D top-down avec progression RPG légère.

## Boucle principale
Explorer une zone → faire quêtes secondaires → trouver la carte → entrer dans le donjon → résoudre énigmes → battre le boss → débloquer une capacité → accéder à la suite.

## Histoire
Le héros vit dans un village paisible. Pendant le tutoriel, le village est attaqué par le grand méchant. Sa famille meurt, le village est pillé, et le héros survit. Il devient vagabond et cherche à devenir assez fort pour se venger.

## Progression
Le héros ne gagne pas d'arbre de compétences complexe. Il monte de niveau :
- HP
- attaque
- défense

L'équipement se trouve surtout dans les magasins :
- épées
- boucliers
- armures
- potions

## Coffres
Les coffres donnent :
- pièces
- potions
- clés
- fragments de lore
- cartes de zone

## Zones
1. Bois des Cendres
2. Marais Noir
3. Falaises du Vent
4. Désert des Mirages
5. Mines d'Acier
6. Catacombes Oubliées
7. Forteresse de Glace
8. Royaume Corrompu
9. Sanctuaire du Néant
10. Forteresse du Tyran

## Capacités
Chaque fin de donjon débloque une capacité utilisable ensuite.
Pour éviter un scope trop grand, commencer par 4 capacités principales :
- Shadow Dash
- Dark Burst
- Void Pull
- Rift Step
